# Task: Risk Areas & Gaps (Insight #10)

## Goal

Identify areas where a candidate falls short of job requirements, framed constructively with severity, context, and mitigation suggestions. Designed for hiring managers.

## Scope

**In scope:**

- AI module (schema, prompt, compute) following the established insight pattern
- Display component with severity-coded risk cards
- Tier gating: Free users see top 3 risks, Pro users see all
- On-demand generation only (registered users, via Generate button)
- Registration in all integration points + homepage showcase

**Out of scope:**

- Public/guest access
- Automatic computation during matching

## Acceptance Criteria

- [ ] AI identifies risk areas with severity (critical, moderate, minor)
- [ ] Each risk includes detail and optional mitigation suggestion
- [ ] Free users see top 3 risks + upgrade CTA, Pro users see all
- [ ] Available only in dashboard via "Generate" button (business/hiring manager tab)
- [ ] Card enabled on homepage insights section
